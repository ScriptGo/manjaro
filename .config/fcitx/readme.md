## 配置文件存放文件

Windows: %APPDATA%\Rime

Linux  : ~/.config/fcitx/rime

## 主要配置文件

installation.yaml # 同步配置文件和词库

default.custom.yaml # 全局配置--设定备选词数量，定义输入方案

luna_pinyin_simp.custom.yaml # 明月拼音相关配置--定义扩充词库、加载符号库、模糊拼音

symbols.custom.yaml # 存放自定义符号
